🚀 BƯỚC TIẾP THEO:

1. IMPORT DATABASE:
   - Truy cập: http://localhost/phpmyadmin
   - Chọn "Nhập dữ liệu" (Import)
   - Chọn file: c:\xampp\htdocs\HTTTQL\database.sql
   - Bấm "Thực thi"
   
   HOẶC dùng PowerShell:
   cd C:\xampp\htdocs\HTTTQL
   mysql -u root -p < database.sql
   (Nhấn Enter khi hỏi password)

2. TEST KẾT NỐI:
   - Truy cập: http://localhost/HTTTQL/test_database.php
   - Kiểm tra tất cả bảng được tạo và dữ liệu

3. ĐĂNG NHẬP VÀ TEST:
